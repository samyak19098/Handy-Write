ABOUT
Handy- write is an interactive writing platform developed to aid parents in teaching their children writing. Using its kid- friendly interface and an interactive environment, it is the ideal tool for your kid to learn and practice writing. It is also accompanied with an improved vocabulary. We use a third party whiteboard - Air Sketch - to project the writing on a
computer. We use Tesseract.js to identify the written text. It is a JavaScript port for the Tesseract library developed for Java. A striking feature of Handy- write is the use of an LDR, active buzzer and touch sensor to make the platform not only more interactive but also allowing the kid to work in a safe environment by ensuring he/she is working in proper lighting conditions. Accessibility is another thing that we kept in mind while designing the platform. Each webpage has an option of enabling voiceover which helps the visually impaired in writing just as good as everyone else.
